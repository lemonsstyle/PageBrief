本隐私说明适用于浏览器扩展“PageBrief”。该扩展的单一用途是在用户主动点击后，对当前网页生成摘要。

1. 插件会处理哪些数据

  • 当前网页的标题、URL、描述、正文片段，或用户当前选中的文本。
  • 用户填写的模型配置，包括模型服务模板、API URL、Model ID 与 API Key。
  • 插件本地生成的摘要结果、最近 10 条摘要缓存与调试日志。

2. 数据在什么情况下会发送出去

  • 只有当用户主动点击“总结当前页”时，插件才会把网页内容发送到用户选择的模型服务。
  • 插件不会把网页内容发送给本扩展开发者自己的服务器；当前版本不包含开发者中转后端。
  • 如果用户选择的是第三方模型服务，例如 OpenAI、DeepSeek、OpenRouter 或 SiliconFlow，则数据会直接发送到对应服务提供方。
  • 如果用户选择的是本地模型接口，例如 Ollama、LM Studio 或其他本机 OpenAI 兼容服务，则数据只会发送到本地地址。

3. 数据会存在哪里

  • 最近摘要与调试日志保存在当前浏览器的本地扩展存储中。
  • 模型配置仅会在用户点击保存配置后写入当前浏览器本地，以便下次继续使用。
  • 开发者不会自动接收这些本地存储内容。

4. 数据用途

  • 读取当前网页内容，仅用于生成摘要。
  • 保存模型配置，仅用于减少重复输入。
  • 保存最近摘要与调试日志，仅用于提升使用体验与排查错误。

5. 数据共享

  • 本扩展不会出售用户数据。
  • 本扩展不会将用户数据用于广告定向。
  • 本扩展仅会在用户主动操作时，将当前网页内容发送给用户自己选择的模型服务提供方。

6. 用户可以如何控制数据

  • 用户可以不填写 API Key，仅使用本地模型接口。
  • 用户可以修改或清空本地保存的模型配置。
  • 用户可以在插件界面中清空本地调试日志。
  • 用户可以卸载扩展，以移除浏览器本地扩展存储中的相关数据。
